# Edmacro for pet catchers
## Requirements:
### Roblox settings:
 - shift lock: disabled
 - camera mode: classic
- camera sensitivity: 1
- full screen: off
- graphics quality: 1(minimum)
- movement mode: default (keyboard)
- reduce motion: off

### In-game settings:
- chat: off
- render distance: low
- UI scale: 1
- hide your pet: on
- hide other pets: on
- disable player eggs: on
- stats: off (not required but recommended)
